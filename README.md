## Guide for My Portfolio

The `Case_Studies` folder contains all case studies that I have completed and each one is independent of each other and showcase different skills. For example, the Salifort Motors case study displays the skills of Machine Learning, Feature Engineering and Model Selection whereas the stock market data case study shows my skills in web scraping and parsing HTML data.

`Cumulative_Case_Studies` are comprehensive case studies that span the course of months to weeks, and build on top of each other respectively. These are from the Google Advanced Data Analytics Professional Certificate and showcase EDA, Statistics and Machine Learning as well as the TikTok case study including Natural Language Processing.

The folder for all my `Kaggle_Competitions` contains all the notebooks used for kaggle competitions, with the same notebooks being available on my kaggle profile.

`Publications` contains all of my publications that I have produced, with these being used for various applications, such as demonstrating knowledge to a study guide or a resource for teaching others concepts.

`Web_Applications` is where all my web applications are contained, here you will find the information about the app and the app files. The app files being files such as the server, templates, static, testing and packages as per the usual structure of a web application.

## Formats

The notebooks will be in either Jupyter if Python is used or R Markdown if R is used, with all notebooks being available in markdown. This is because when viewing notebooks on GitHub, if you are using a smartphone, the Jupyter files will not render and will not be available to view but the markdown files will be viewable. All microsoft word documents will be in pdf format, all images in png format and all excel documents will be in csv format.
